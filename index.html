This is the portfolio of Alan, a full-stack web developer based in Honolulu, Hawaii.

These five projects provide the best sampling of my skillset: UI development, 2D game art, Node.js, Angular.js and ReactJS, SQL database creation, algorithm visualization, and raw Javascript.

Pixel Painter

A simple paint tool for the browser. Includes a fill bucket using breadth-first search, a shareable URL generator for pictures made with the tool, and a button to revert from last save.

Sorting Visualizer

A visualization tool for common sorting algorithms made with pure CSS3/HTML5/JavaScript. Watch in slow-motion how computers process arrays to be sorted.

Express Gallery

A photo gallery site made with Express, Pug, Redis and PostgreSQL. Users can post new photos or edit/delete their previous posts.

React Kanban

An interactive kanban board made with React, Redux, Redis, Express and PostgreSQL. Create to-do lists and assign them by priority and by user.

SpellCasters

A fast-paced game to teach spelling to kids and adults. Challenge hundreds of commonly misspelled words as well as more difficult vocabulary from the SAT and SSAT standards. Created with AngularJS, Redis, Express and PostgreSQL.
